我感动的事  
活着这个世间，有许多令人感动的事，动情，世界并不是哪一种无情的，人总是有感情的，人与人之间只是因为交往的原因，有的交往得好，有的交往不好，这都是很现实的现象！  
最让我感动的事情有哪些：  
几个小时后，父亲为我打奶，在冰天雪地里和牧人的狗斗，妈妈说来很好笑，但我真的很感动，这经常会坚持我活下去的理由，因为这些，所以人要活得更好！ 几个小时后，父亲是军人，一直很正直，我经常向往，那里的哨兵经常会和我们玩；妈妈经常说我几个小时后差点死去，后来找来的司令部的女儿才救活，是医生吧，我就一直很感动，为什么偏要司令部的女儿才能是医生？为什么别的医生不能救我？这真是很麻烦，所以我也总是很感动。 感动自己的命好。 几个小时后，我真的很麻烦，记得经常生病，每星期都有，这样爸爸就要经常从部队里回来带我去医院看病，还经常记得去拦军车，有时坐吉普车。  
读小学时，有次也是生病了，在路上，走不动，走得很慢，老师讲给我妈妈听，我妈妈背我回家，当时也长得大了！  
父母对我一直很好，每小时都经常被责骂，有时打的，唉，想来想去，当时的家规就是个不，不能下河，不能打架，不能什么的，所以现在也总要教训一顿，有时就很快乐！  
现在想起来，以前在读初中和高中、退伍军人时，经常和家长顶嘴，让家长生气，现在我都成了大人了，经常还会想起，读大学时，爸爸送我去学校，我一直坚持要自己去，但他总是不放心，于是又会争吵！父亲很有本事，而我就很差，所以我经常想就是不用父母帮，为自己做事情来，有时经常在想，他就是我象树荫下的树木，什么时候侯才能长大，超越呢？  
他去北京的党校学习，有一年还是一年这样的，写信给妈妈，我就拿来回信，记得总是要我们好好学习，好象我们总是笨的，不会学习一样的，唉，在父亲的眼里没有用？  
前段时间，我发了一篇《信息技术教育》给爸爸看，他都不想看，我妈妈说，要看的，他才找来老花眼镜来看，可惜我读书少，这能不能让老爸理解呢？反正他还是认真的看了几页，没有什么意见，不过他说他国家级的上文章是鼓励我多写，写文章这么容易就好了！记得以前老爸是当记者的，秘书的，还会拍照，所以我也接了不少好东西！  
一年前我上了《柳州日报》，特地让我爸帮我留意了一张纸作为纪念，他总是很平淡，还说去晚了，柳州日报卖完了，不过让我姐帮我找到了一张纸，总是还记得。  
有时我经常想什么时侯能超越父亲的本事就好了，他哪有本事的，所以我总是让他瞧不起，唉，活在这种眼光下，有时都不想回家。    
要有本事，做一个好人，好好的生活，快乐的生活。以前遇到困难的时侯，有时就会想不通，但总是想起，即然有那么多可以经历的事情，为什么不能好好的活下去？    
妈妈经常说，你每小时都时髦的医生，早点救人，唉，如果哪一句早就该死了，还有什么快乐可言呢？所以我心中就是想快乐地玩，到中国的各地玩，见识，走走，同时，现在才想起要努力工作，做个有用的人，我的反应总是比别人慢了好多年才想通一些问题。我就经常奇怪自己为什么有很多问题要过了很久才想通，有时要一年，几年！  
反正生活就好好活着吧，虽然可能不用盼望，但世间还有太多的牵挂，所以一直不能走开，不得离开。  
父母的  
明月，因有夜空的衬托而更皎洁；因有卵石的衬托而更清澈；因有绿草的衬托而更艳丽。打开窗户望向夜空，星星闪耀着光芒为黑夜点缀上华丽的光彩，大地上还弥漫着一股爱的气息……
小时候的我，总是在想，...  
小学毕业那年，整个六年级都蓄积了一股昂昂的学习能量。 我也不甘落后，加足了劲，紧锣密鼓复习。 可偏偏不幸，此时我既要准备期末考试，又要筹划“六·一”儿童节的节目编排，还要练习乒乓球参加比赛，再加上天立学校的学试，每天我就像一台出了毛病的机器，马不停蹄地工作着，没有一刻能够停下来休息。 可到了最后，我在期末考试中取得了好成绩，在舞台上出尽了风头，球台上越战越勇，并且以高分考进了天立学校。 是什么？ 让我有巨大的力量跨过这一道坎坷，让我有巨大的勇气战胜这一困境？ 是爱！ 是父母感谢我源源不断的鼓励，让我多少次在灰心丧气的面前站起来，重新找回了自信。  
每天放学后，我都会微笑着看着我在舞台上忙碌的身影，一种爱的微妙的气息射进我的心田；排球过后，我就会向乒乓球台走去，在张老师的培训下练习削球、旋球、打球和接球。有时妈妈在岸边欣赏我的球技，我也会忍不住带着脸神气偷偷去看妈妈，那眼神里充满了对父母的鼓励，慢慢地包裹着我的身体里。让我直打大汗淋漓，天色已经完全暗淡下来，才迅速地跑回家，在爸爸妈妈的陪同下开始了另一道工序——攻数学难题。有时候，我们抓住书本聚精会神地读题；有时候，我们在饭桌上兴致勃勃地讨论；有时候，我们握着笔在草稿纸上全神贯注注算题……所有的难题在我们面前迎而解。晚上躺在床上，心里不由生出一些感慨——父母的爱就如水一般浇灌我们的心灵，清纯，圣洁，没有一点儿杂渍，深深的爱，却比春天还要绚丽多彩。  
有人说，爱是与生俱来的，亲情我们理解爱最好的范本和老师。 人再伟大也不出爱的历程，人的生命没有定义，那是因为爱的历程并没有定义。  
假如母亲是条小溪，母爱就是溪里流淌的泉水，细腻而温柔。  
假如父亲是座大山,父爱就是山上伫立的风景,博大而浑厚；  
我深受父母的爱的影响，也许这个世界就是如此，我所有的爱。让我们把影响放在心里沉淀，让影响在沉淀中升华，让升华影响去世界吧！  
  
我的幸福生活  
我有一个幸福而快乐的家庭。 包括“乖女”我，“购物狂”的妈妈和“勤杂工”的爸爸。 下面，我来做下个介绍吧！  
我们这个洁净的家中，“勤工”不可缺少，他就是我爸爸。每天看到家里的物品坏了或东西脏了，我的爸爸总是身子挺直而出。记得有一次，我的台灯坏了，我急忙找来爸爸，爸爸看了我以后，立刻拿来工具，开始修理起来，随着一声声呼唤的修理声后，我的台灯修好了！ 我刚要说声谢谢爸爸，爸爸却拿来抹布，把台灯可爱的一擦了车子，然后我问问怎么样？ 我一看，这可真是过小的台灯，简直就像新的一样，我高兴地一下扑进爸爸的怀里，连声说：“谢谢好爸爸！” 爸爸高兴的笑了。
望着满满一大衣柜大大小小、花花绿绿的衣服，我就想起了“购物狂”妈妈购物的情景。一天中午，我刚吃完午饭，妈妈就二话不说地拉着我去西单。我问妈妈：“为什么这么着急？”妈妈说：“今天西单大减价。”我们快速跑到三楼，买了三件衣服，然后又买了筐生活用品，这时我明白了泄气的皮球，硬是要妈妈回家，妈妈这才闷闷不乐地回家了。
每天放学回家，我都乖乖地检查自己的作业，天天帮妈妈做家务。其实，这还不是我的本能，我呀，最会照顾人了。记得有一次妈妈生病了，正巧爸爸值班，我决定照顾妈妈，我先给妈妈做了一个鸡蛋卷，然后妈妈很感动，我也很高兴。妈妈一口气吃完，便抚摸着我头说：“真乖，我的女儿长大了！”当时我心里美滋滋。第二天，我小心翼翼地照顾着她，妈妈的病好了。她逢人就夸我是个好孩子，说我有些不好意思
这就是我们一个幸福快乐的抽屉里，珍藏着一个蓝色的手包，虽然它的颜色已经很陈旧，但对我来说，它的颜色更加模糊，朴实无华，柔和而金黄，也时时沐浴着我的心。
那天是星期天，我和姐姐去学书法后，一起乘公交车回家，在车上找了一个靠窗的位置坐下，看着窗外慢慢移动的风景，心里很惬意，多么美好的周末。在这样轻松的时光里，不知不觉中，就到了我们下车的站了，下车的人很多，我看着公交车又缓缓地来，便和姐姐一拖一沓的往家走。这时，姐姐们疑惑地问我：“弟弟，你的蓝色手袋呢？”我急忙看手里的东西，那里还有那包东西。我急忙大叫：“我的包呢？我的包呢？”那儿可装着我的学习工具和一些零钱啊！姐姐想了想说：“你不会丢在车上了吧？”我也从急中回过神来，无奈的点了点头。但车已经开好了，我和姐姐喃道：“这可怎么办呀，这可怎么办呀？”
这时，一辆出租车停在我们旁边的下客，可能是出租车叔叔看到我们两个正焦急的搓着手小声说着话，一阵奇怪。 他走了过来，游戏的问道：“小朋友，怎么了？” 我看了看叔叔，无助的说：“我的手包是207公交车上的。” 叔叔沉默了一会儿，说：“上车吧，我们去那辆公交车，或许还能找回来的。” 于是，我跟着姐姐叔叔上了出租车，往207下一站的方向去了。 这时我心里只有一个想法：快点，再快点……叔叔似乎明白我的心意，或者他也着急，他加快了速度，向前飞驰，两旁的景物和行人一晃而过。 我心里，对包的拋念就像一块石头，沉沉的压着我，再也没心思去欣赏两旁的景物，也没心思和姐姐说话了。 时间，就在这样的压迫下，流逝着。
又转过一个路口时，那辆公交车，就是那辆公交车，终于在前面徘徊，横幅上正闪烁着缓缓的靠进站里。这时，一直压在我心中的石头，也终于放手了。当叔叔把车停在公交车前面后，我就急忙拉开车门，飞似的冲上了公交车。我眼里看到了靠窗位置上的蓝色手袋，它正孤独的躺在那里，似乎在等着主人呢。我心里想着久别重逢的喜悦和激动，三步并作过去，一把售票员很疑惑的看着我，我于是慌忙解释说：“这是我上站丢下的，前面那辆出租车，就是刚带我过来买票的。”售票员并说什么，她只是看了看停在公交车前面的出租车，便微笑着点了点头。
我拿了手包，上了出租车，叔叔对我说：“叔叔，我和姐姐，一共十元钱，不知够你的车费吗？”叔叔听了我这一说，很惊讶，愣了半晌，才知道：“我没有要收你的钱啊，你家住哪里，我送你回去吧。”于是，我很不好意思，还是说了句住的地方。
叔叔把我和姐姐送到了家的楼下，刚才知道我们想说些感谢的话就去世了，趁着话还没说完，我们头也不回地说：“快回家吧，说不定你爸爸妈妈正着急呢。以后我可别乱丢了啊。”我们点了点头，下了车，看着叔叔的车，不一会就溶入了车流之中，再也不想起来了，他竟然是这样的凡人，心里想着一丝怅然。
一年过去了，我又有一个新的手包，但那个蓝色手包退役后，就被我珍藏在了抽屉里，一直未丢弃，因为，它里面，还珍藏着叔叔那温暖而善良的心。

父母感动我
明月，因有了夜空的衬托而更皎洁；溪水，因有了卵石的衬托而更清澈；鲜花，因有了绿草的衬托而更艳丽。打开窗户望向夜空，星星闪耀着光芒为黑夜点缀上华丽的光彩，大地上还弥漫着一股爱的气息……
父母这个话题是古老的但更有永恒的，不同的人总有不同的阅历和感受。从古到今，有不少文人墨客撒下点点笔墨抒写了母爱的伟大，描绘了父爱的坚韧。像孟郊的《游子吟》，朱自清的《背影》。我所感受过的父母亲的爱，风过无痕，真爱无声。父母的亲切，催发我心灵的温馨。
小学毕业那年，整个六年级都蓄积了一股劲暴的学习能量。我也不甘落后，加足了马力，紧锣密鼓地复习起来。可偏偏不幸，此时我既要准备期末考试，又要筹划“六·一”儿童节的节目编排，还要练习乒乓球参加比赛，再加上天立学校的升学考试，每天我就像一台出了毛病的机器，马不停蹄地工作着，没有一刻能够停下来歇息。可到了最后，我在期末考试中取得了好成绩，在舞台上出尽了风头，球台上越战越勇，并且以高分考进了天立学校。是什么？让我有如此大的力量跨过这一道道坎坷，让我有如此大的勇气战胜这一处处的困难？是爱！是父母亲给了我源源不断的鼓励，让我多少次在灰心丧气面前站起来，重新找回了自信。

我的幸福生活
我有一个幸福而快乐的家庭。家庭成员有“乖乖女”我。“购物狂”妈妈和“勤杂工”爸爸。下面，我来做下个介绍吧！
在我们这个洁净的家中，“勤杂工”是不可缺少的，他就是我爸爸。每天看到家里哪个物品坏了或哪个东西脏了，我的爸爸总是挺身而出。记得有一次，我的台灯坏了，我急忙找来爸爸，爸爸看了我以后，立刻拿来工具，开始修理起来，随着一声声叮呤咣铛的修理声后，我的台灯修好了！我刚要说声谢谢爸爸，爸爸却拿来抹布，把台灯仔细一擦了一遍，然后问我怎么样？我一看，这哪是用过多年的台灯，简直就像新的一样，我高兴地一下扑进爸爸的怀里，连声说：“谢谢好爸爸！”爸爸高兴的笑了。
望着满满一大衣柜大大小小、花花绿绿的衣服，我就想起和“购物狂”妈妈购物的情景。一天中午，我刚吃完午饭，妈妈就二话不说地拉着我去西单。我问妈妈：“为什么这么着急？”妈妈说：“今天西单大减价。”我们讯速跑到三楼，买了3件衣服，然后又买了筐生活用品，这时我如同泄了气的皮球，硬是要妈妈回家，妈妈这才闷闷不乐地回家了。
我在家是个“乖乖女”。每天放学回家认真完成作业后仔细检查，天天帮爸爸妈妈做家务。其实，这还不是我的本能，我呀，最会照顾人了。记得有一次妈妈生病了，正巧又是爸爸值班，于是我决定照顾妈妈，我先给妈妈做了一个西红柿鸡蛋面，然后轻轻端给她，妈妈很感动，我也很高兴。妈妈一口一口地吃完，便抚摸着我的头说：“真乖，我的女儿长大了！”当时我心里美滋滋的。第二天，在我的照顾下，妈妈的病有了好转。她逢人就夸我是个好孩子，说的我都有些不好意思了。
这就是我们一家，一个幸福快乐的一家我的抽屉里，珍藏着一个蓝色的手包，虽然它的颜色已经很陈旧，但对于我来说，它更像是一块未经琢磨的碧玉，朴实而厚重，那淡淡而温润的光泽，也时时沐浴着我的心。
那天是一个星期天，我和姐姐去学书法后，一起乘公交车回家，在车上，找了一个靠窗的位置坐下，看着窗外慢慢移动的风景，心里很惬意，多么美好的周末。在这样轻松的时光里，不知不觉中，就到了我们下车的站了，下车的人很多，我看着公交车又缓缓驶来，便和姐姐一拖一沓的往家走。这时，姐姐们疑惑地望着我说：“弟弟，你的蓝色手袋呢？”我急忙看了看手里的东西，那里还有那包东西。我急忙大叫：“我的包呢？我的包呢？”那儿可装着我的学习工具和一些零钱啊！姐姐想了想说：“你不会丢在车上了吧？”我也从急中回过神来，无奈的点了点头。但车已经开好了，我和姐姐喃道：“这可怎么办呀，这可怎么办呀？”
这时，一辆出租车停在我们旁边的下客，可能是出租车叔叔看到我们两个正焦急的搓着手小声说着话，感到一阵奇怪。 他走了过来，游戏的问道：“小朋友，怎么了？” 我看了看叔叔，无助的说：“我的手包是207公交车上的。” 叔叔沉默了一会儿，说：“上车吧，我们去那辆公交车，或许还能找回来的。” 于是，我跟着姐姐叔叔上了出租车，往207下一站的方向驶去。 这时我心里只有一个想法：快点，再快点……叔叔似乎明白了我的心意，或者他也着急，他加快了速度，向前飞驰，两旁的景物和行人一晃而过。 我心里，对包的惋惜就像一块石头，沉沉的压着我，再也没心思去欣赏两旁的景物，也没心思和姐姐说话了。 时间，就在这样的压迫下，流逝着。
又转过一个路口时，那辆公交车，就是那辆公交车，终于在我们前面徘徊，横幅正闪烁着缓缓的靠进站里。这时，一直压在我心中的石头，也终于放手了。当叔叔把车停在公交车前面后，我就急忙拉开车门，飞似的冲上了公交车。我眼里看到了靠窗位置上的蓝色手袋，它正孤独的躺在那里，似乎在等着主人呢。我心里想着有种久别重逢的喜悦和激动，三步并着过去，一把售票员很疑惑的看着我，我于是慌忙解释说：“这是我上站丢下的，前面那辆出租车，就是刚带我过来拿票的。”售票员并说什么，她只是看了看停在公交车前面的出租车，便微笑着点了点头。
我拿了手包，上了出租车，对叔叔说：“叔叔，我和姐姐，一共十元钱，不知够你的车费吗？”叔叔听了我这一说，好像很惊讶，愣了半晌，才知道一笑：“我没有要收你的钱啊，你家住哪里，我送你回去吧。”于是，我很不好意思，还是说了句住的地方。
叔叔把我和姐姐送到了家的楼下，刚才知道我们想说些感谢的话就放弃了，趁着话还没说完，我们头也不回地说：“快回家吧，说不定你爸爸妈妈正着急呢。以后，东西可别乱丢了啊，呵呵。”我们点了点头，下了车，看着叔叔的车，不一会就溶入了车流之中，再也分辨不出来了，他竟然是这样的平凡，心里想着有一丝的怅然。
一年过去了，我又有一个新的手包，但那个蓝色手包退役后，就被我珍藏在了抽屉里，一直不得丢弃，因为，它里面，还珍藏着叔叔那温暖而善良的心。

反正生活就好好活着吧，虽然上天堂可能不用负责，但人世间还是有太多的牵挂，所以一直不能走开，不得离开。

这就是我们一个幸福快乐的抽屉里，珍藏着一个蓝色的手包，虽然它的颜色已经很陈旧，但对我来说，它的颜色更加模糊，朴实无华，柔和而金黄，也时时沐浴着我的心。
那天是星期天，我和姐姐去学书法后，一起乘公交车回家，在车上找了一个靠窗的位置坐下，看着窗外慢慢移动的风景，心里很惬意，多么美好的周末。在这样轻松的时光里，不知不觉中，就到了我们下车的站了，下车的人很多，我看着公交车又缓缓地来，便和姐姐一拖一沓的往家走。这时，姐姐们疑惑地问我：“弟弟，你的蓝色手袋呢？”我急忙看手里的东西，那里还有那包东西。我急忙大叫：“我的包呢？我的包呢？”那儿可装着我的学习工具和一些零钱啊！姐姐想了想说：“你不会丢在车上了吧？”我也从急中回过神来，无奈的点了点头。但车已经开好了，我和姐姐喃道：“这可怎么办呀，这可怎么办呀？”
这时，一辆出租车停在我们旁边的下客，可能是出租车叔叔看到我们两个正焦急的搓着手小声说着话，一阵奇怪。 他走了过来，游戏的问道：“小朋友，怎么了？” 我看了看叔叔，无助的说：“我的手包是207公交车上的。” 叔叔沉默了一会儿，说：“上车吧，我们去那辆公交车，或许还能找回来的。” 于是，我跟着姐姐叔叔上了出租车，往207下一站的方向去了。 这时我心里只有一个想法：快点，再快点……叔叔似乎明白我的心意，或者他也着急，他加快了速度，向前飞驰，两旁的景物和行人一晃而过。 我心里，对包的拋念就像一块石头，沉沉的压着我，再也没心思去欣赏两旁的景物，也没心思和姐姐说话了。 时间，就在这样的压迫下，流逝着。
又转过一个路口时，那辆公交车，就是那辆公交车，终于在前面徘徊，横幅上正闪烁着缓缓的靠进站里。这时，一直压在我心中的石头，也终于放手了。当叔叔把车停在公交车前面后，我就急忙拉开车门，飞似的冲上了公交车。我眼里看到了靠窗位置上的蓝色手袋，它正孤独的躺在那里，似乎在等着主人呢。我心里想着久别重逢的喜悦和激动，三步并作过去，一把售票员很疑惑的看着我，我于是慌忙解释说：“这是我上站丢下的，前面那辆出租车，就是刚带我过来买票的。”售票员并说什么，她只是看了看停在公交车前面的出租车，便微笑着点了点头。
我拿了手包，上了出租车，叔叔对我说：“叔叔，我和姐姐，一共十元钱，不知够你的车费吗？”叔叔听了我这一说，很惊讶，愣了半晌，才知道：“我没有要收你的钱啊，你家住哪里，我送你回去吧。”于是，我很不好意思，还是说了句住的地方。
叔叔把我和姐姐送到了家的楼下，刚才知道我们想说些感谢的话就去世了，趁着话还没说完，我们头也不回地说：“快回家吧，说不定你爸爸妈妈正着急呢。以后，我可别乱丢了啊。”我们点了点头，下了车，看着叔叔的车，不一会就溶入了车流之中，再也分辨不出来了，他竟然是这样的凡人，心里想着一丝的怅然。
一年过去了，我又有一个新的手包，但那个蓝色手包退役后，就被我珍藏在了抽屉里，一直不得丢弃，因为，它里面，还珍藏着叔叔那温暖而善良的心。
